 📌 Objective

The aim of this project is to build a **To-Do List web application** that allows users to manage daily tasks by adding new tasks, viewing pending tasks, marking tasks as completed, and clearing completed items. This project strengthens understanding of **HTML, CSS, and JavaScript** for building interactive web applications.

 🛠️ Tools & Technologies Used

* HTML – For website structure
* CSS – For layout and visual styling
* JavaScript– For interactive functionality and task management
* Web Browser – To run and test the app
* GitHub Pages– To publish the live web app

 🔧 Steps Performed

1. Designed the webpage layout (input area, task list sections).
2. Created input field and buttons using HTML.
3. Styled the UI for better visuals with CSS.
4. Added JavaScript to handle:

   * Adding new tasks
   * Marking tasks as pending or completed
   * Tracking task status
   * Clearing completed tasks
5. Tested app functionality and user interactions.
6. Deployed live using GitHub Pages.

 ✅ Outcome

A functional **To-Do List Web App** that:

* Lets users add tasks and view them in a list
* Shows counts for pending and completed tasks
* Separates pending and completed tasks
* Allows clearing of all completed tasks
* Is accessible online via GitHub Pages
